Get RubyMC

There are 3 ways in which you can get RubyMC:

Build a server

 You will be taken through an online wizard-style setup.
This offers a great degree of customisation for your server, with minimal difficulty.
Difficulty: Easy

Download a pre-configured server

 All you need to do is specify name, motd and port.
The rest is done for you with the most common options selected.
This option is the fastest if you want to get your server up and running quickly.
Difficulty: Easy

Download a minimal-config server

 You will need to edit text-based configuration files by hand.
This offers high customisability but is difficult.
Difficulty: Hard
